Summary of terms

The ESSIM Vocabulary defines one the following terms.

Class essim:Field
label Data field
comment Data field in a Social Security form.
Class essim:Relation
label Relation
comment Relation between a data field and a term in the Glossary of Terms.
Class essim:Language
label Language
comment Representation of names of languages.
subClassOf iso639-1:iso639-1_Language
subClassOf dcterms:ISO639-2
subClassOf dcterms:ISO639-3
Datatype property essim:dateCreated
label Date Created
comment .
domain essim:Field
range rdfs:Literal
subPropertyOf dc:date
Datatype property essim:dateModified
label Date Modified
comment .
domain essim:Field
range rdfs:Literal
subPropertyOf dc:date
Object property essim:creatorInstitution
label Creator Institution
comment ..
domain essim:Field
range ir:Institution
